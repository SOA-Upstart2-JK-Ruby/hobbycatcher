# Facebook API Client

Gather useful information from Facebook pages API.

## Resources

- PageInfo
- Reviews
- Posts

## Elements

- PageInfo
  - id
  - name
  - category
  - profile picture
  - followers count
  - rating
  - website
  - location
  - about
- Reviews
  - sentiment (positive/negative)
  - review
- Posts
  - id
  - date
  - content
